# 學生使用手冊模板

> 老師可在此填入正式系統網址：`請填入網址`

## 1. 登入

請使用自己的 Google 帳號登入。請勿使用同學或家人的帳號。

## 2. 上傳作業

進入「上傳作業」，填寫姓名、學號、Gmail、班級、實習機構與實習機構督導姓名，選擇作業種類後上傳檔案。

## 3. 查看我的紀錄

上傳後請到「我的紀錄」查看狀態。若剛上傳後還沒有看到紀錄，請點選「重新辨識並讀取紀錄」。

狀態說明：

- 已繳交：系統已收到檔案，正在等待老師檢閱。
- 已完成：老師已確認該項作業完成。
- 退件：老師要求修正，請依退件原因重新上傳。
- 老師回傳：老師若有上傳批閱後檔案，會在此欄位顯示。

## 4. 訪視安排

請先與實習機構確認可訪視日期、時間、地址與聯絡人，再到「訪視安排」填寫回報。
